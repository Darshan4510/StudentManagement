Student Management System

This project is a basic Student Management System built using ASP.NET and C#.
It allows you to manage student details with simple operations.

Features:

Add student

Edit student

Delete student

View student list

Technology Used:

ASP.NET (C#)

HTML, CSS, JavaScript

Visual Studio

Project Overview:
The project contains pages and code needed to handle student information using a clean and easy structure. It is suitable for beginners learning ASP.NET or for building basic CRUD applications.

How to Run:

Open the solution file: StudentManagement.sln

Build the project in Visual Studio

Run the application (F5)
